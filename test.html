📌 نکات مهم:

  • از کلاس‌های transition-all duration-300 برای نرمی حرکت استفاده شده.
  • افکت موج (ریپل) با CSS اضافی پیاده‌سازی شده و در بقیه موارد فقط Tailwind کافی است.
  • می‌توانید مدت زمان (duration-200, 300, 500, 700) و نوع منحنی (ease-out, ease-in-out) را تغییر دهید.
  • برای چرخش از hover:rotate-180، برای مقیاس hover:scale-110 و برای حرکت hover:-translate-y-1 استفاده شده.
